# Онлайн чат rest api

Веб-приложение, типа Telegram, которое пользователь может запустить, и присоединиться к другим пользователям, чтобы пообщаться с ними, отправляя им текстовые сообщения.

## Stack
+ Spring core
+ Spring Security
+ Spring data mongodb
+ Spring WebSockets
+ MongoDB
+ Yandex Clod + AWS S3
